# Java API Labs

Coleção de projetos desenvolvidos para praticar consumo de APIs REST com Java.

## Projetos

### Book Search
Aplicação Java que consome a API pública do Google Books para buscar informações de livros a partir do título informado pelo usuário.

### Crypto Search
Aplicação desenvolvida em Java que consome a API pública da CoinGecko para buscar informações sobre criptomoedas a partir do nome informado pelo usuário.

## Tecnologias

- Java 17
- HttpClient
- REST APIs
